# Android Note App

Welcome to the Android Note App project! This app provides a simple and intuitive way for users to take and organize notes on their Android devices. Whether you're a student, professional, or just someone who likes to jot down thoughts, this app is designed to make note-taking easy and efficient.

## Features

- **Create and Edit Notes:** Write down your thoughts, ideas, or important information with ease. Edit your notes whenever you need to make changes.

## Getting Started

To get started with the Android Note App, follow these steps:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/HienDao14/NoteApp.git
2. **Open in Android Studio:**
- Open Android Studio.
- Click on "Open an existing Android Studio project".
- Navigate to the cloned repository and select the project.
3. **Build and Run:**
- Build the project using Android Studio.
- Run the app on emulator or a physical Android device.
4. **Explore the App:**
- Familiarize yourself with the features and functionalities of the app.

Thank you for using the Android Note App! Happy note-taking!
